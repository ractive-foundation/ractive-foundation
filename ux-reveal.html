 Semantic Event Mapping
toggleModal - Opens or closes the modal box, depending on the box's current visible state.


Semantic Data Model
text - The text to display on the link that opens the modal box.
class - The class for the aforementioned link.
size - Allows for the use of foundation size styles on the modal.
content - The HTML content that is to be displayed in the modal box.
closetext - The text that will be rendered as raw HTML within the modal's 'close button' element
